# ChatGPT記事自動生成エージェント (Article Generator Agent)

**Writer (Gemini 2.5 Pro)** と **Simulator (GPT-4o)** という2つのAIが連携し、実践的な「ChatGPT活用ガイド記事」を全自動で執筆するツールです。

Writerが記事の構成とプロンプトを考え、Simulatorがそのプロンプトを実際に実行して回答を生成します。これにより、「やってみた」形式の説得力ある記事がボタン一つで完成します。

## ✨ 特徴

*   **全自動執筆**: テーマを決めるだけで、導入からまとめまで自動生成。
*   **実演付き**: 実際にChatGPT(GPT-4o)を動かした「生の回答ログ」が記事に含まれます。
*   **構成カスタマイズ**: `format.md` を編集することで、記事のテンプレートを自由に変更可能。

## 🚀 セットアップ

1.  **APIキーの設定**:
    *   `.env.example` をコピーして `.env` にリネームします。
    *   `.env` ファイルに OpenAI と Google (Gemini) のAPIキーを貼り付けます。
    ```env
    OPENAI_API_KEY=sk-...
    GOOGLE_API_KEY=AIza...
    ```

2.  **インストール**:
    *   付属のセットアップスクリプトを実行するだけで、仮想環境の作成とライブラリのインストールが完了します。
    ```powershell
    .\setup_env.bat
    ```

## 📝 使い方

### 1. 記事を生成する
仮想環境を有効化してスクリプトを実行します。

```powershell
# 仮想環境に入る
.\.venv\Scripts\activate

# 実行
python main.py
```
完了すると、フォルダ内に `generated_article.md` が生成されます。

### 2. テーマを変更する
`main.py` を開き、`topic` 変数を書き換えてください。

```python
# main.py 34行目付近
topic = "英語のメールを素早く書く方法"
```

### 3. 記事の構成を変える
`format.md` ファイルを編集することで、記事の見出しや流れを指示できます。
デフォルトでは「導入 → 原因分析 → 解決策（ステップ1〜3） → まとめ」という構成になっています。

## 💡 実用例

このツールは以下のような記事作成に適しています。

### 例1：業務効率化ガイド
*   **テーマ**: 「単純なミス（ケアレスミス）を繰り返す」
*   **生成される内容**:
    *   なぜミスが起きるのかの認知科学的な解説。
    *   ChatGPTに「ミスの傾向」を分析させるプロンプト。
    *   実際の分析結果と、それに基づいたチェックリスト作成手順。

### 例2：スキル習得・学習
*   **テーマ**: 「Pythonでスクレイピングを独学する」
*   **生成される内容**:
    *   初心者が躓きやすいポイントの解説。
    *   ChatGPTに「学習カリキュラム」を作らせるプロンプト。
    *   エラーが出たときのChatGPTへの質問の仕方（実演付き）。

### 例3：クリエイティブ作業
*   **テーマ**: 「魅力的なキャッチコピーを量産する」
*   **生成される内容**:
    *   ターゲット分析のフレームワーク解説。
    *   ChatGPTに「ペルソナ」を設定して案を出させるプロンプト。
    *   出てきた案をブラッシュアップする対話の実例。

## ⚠️ トラブルシューティング

*   **Error: API keys not found**: `.env` ファイルが正しく作られているか確認してください。
*   **Timeout Error**: Simulator (GPT-4o) の応答が遅い場合に発生します。`main.py` の `request_timeout` の値を増やしてみてください。
*   **Gemini Model Not Found**: Google AI StudioのAPIキーが有効か、または利用可能なリージョンか確認してください。
